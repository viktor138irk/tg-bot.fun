# BotFactory v2.2.1 — Restore Design Hotfix

Фронтенд возвращён к исходному тёмному дизайну BotFactory из первого архива.

Сохранены backend-исправления v2.2:
- исправленный запуск platform/admin bot;
- реальные API `/api/admin/*`;
- недельный тестовый тариф;
- индивидуальная постоплата;
- подтверждение/отклонение заказов через API;
- ремонтный скрипт `scripts/repair_server.sh`.

Важно: демо-данные больше не используются как начальное состояние. Панель грузит данные из API `/api/admin/overview`.
Если база пустая, интерфейс будет пустым, а не с фейковыми пользователями/магазинами.
